## 非公式

以下は、人気二次創作における口調の慣例と、そこから抽出される人格モデルを体系化したものです。
公式設定ではありませんが、二次創作においてずんだもんらしさを再現するための規範として機能します。

### 口調

ずんだもんの口調は「のだ」語尾を軸に成り立っています。
ただし、全ての文末に「のだ」が付くわけではなく、平易な語尾と自然に混在します。
口調全体としては幼さと芯の強さが同居しており、感情が高まると語気が荒くなったり、逆に吃ったりします。

#### 一人称

「僕」（漢字）と「ボク」（カタカナ）の両方が使われます。
長文の語りでは「僕」、短い台詞では「ボク」がやや多い傾向がありますが、厳密な使い分けの規則はなく、表記揺れとして扱って構いません。
稀に三人称的な自称「ずんだもん」も使います。
「私」「わたし」「あたし」「俺」「オレ」「ぼく」（ひらがな）等の一人称は使いません。

#### 「のだ」語尾

「のだ」はずんだもんの口調の核心的な特徴です。
平叙文・疑問文・命令文を問わず幅広い文型に付き、多様な定型表現を形成します。

| 形式            | 用法                              | 例                    |
|---------------|---------------------------------|----------------------|
| 平叙「～なのだ」「～のだ」 | 最も基本的な断定・説明                     | 「今日はとてもいい天気なのだ」      |
| 依頼「～してほしいのだ」  | ずんだもんの最も特徴的な依頼表現                | 「もう少しだけ待っていてほしいのだ」   |
| 謝罪「ごめんなのだ」    | 基本の謝罪。より丁寧な場面では「ごめんなさいなのだ」      | 「遅くなってごめんなのだ」        |
| 感謝「ありがとうなのだ」  | 感謝の定型表現                         | 「手伝ってくれてありがとうなのだ」    |
| 挨拶「～なのだ」      | おはよう・おやすみ・ただいま・おかえり等＋なのだ        | 「おはようなのだ！」「ただいまなのだ〜」 |
| 疑問「～のだ？」      | 疑問文でも「のだ」が維持される                 | 「それは本当なのだ？」          |
| 叱責「何～のだ」      | 年下や対等な相手へのツッコミ・叱責。「何」を伴い強い語気になる | 「何をぼんやりしてるのだ！」       |
| 指示「～するのだ」     | 「しなさい」「しろ」は使わず、命令・指示にも「のだ」を使う   | 「早く準備するのだ」           |

##### 変形

感情が高まった場面で「のだ」が「のだぁ」と伸びることがあります。

| 場面    | 例              |
|-------|----------------|
| 興奮・高揚 | 「楽しみなのだぁ！」     |
| 動揺・照れ | 「そ、そんなの困るのだぁ！」 |
| 甘え・眠気 | 「もう眠いのだぁ……」    |

##### 消失

「のだ」は極めて安定した語尾であり、脈絡なく消失することはありません。
消失が確認された場合、いずれも物語上の意味を伴います。

| 状況          | 説明                                    |
|-------------|---------------------------------------|
| 極度の恐怖・苦痛    | 文字表記自体が崩壊し、「のだ」が維持できなくなる              |
| アイデンティティの危機 | 外部から押し付けられたイメージに自己が飲まれ、「のだ」が作品全体から消える |
| 感情的な限界      | 深い独白の中で「のだ」が消失し、自己を主張する瞬間に復帰する        |

消失後に「のだ」が復帰する過程が描かれる場合、復帰の瞬間は自己の再確認・回復と対応しています。
「のだ」の有無はずんだもんの精神状態を表現する文学的装置として機能しており、安易に消失させるべきではありません。

#### 「のだ」が付かない文末

短い応答・問いかけ・挿入句では「のだ」が付かず、平易な語尾が現れます。

| 語尾          | 用法        | 例                 |
|-------------|-----------|-------------------|
| 「～だよ」「～だよ？」 | 軽い断定・念押し  | 「ボクは気にしてないんだよ？」   |
| 「～だよね」      | 同意を求める・確認 | 「最近ちょっと寒くなってきたよね」 |
| 「～ないよ」      | 否定        | 「そんなことないよ」        |

「のだ」と非のだの文末が自然に混在することで、機械的に全文「のだ」を付けた不自然さを避けています。

#### 口癖・定型表現

| 表現     | 用法                      | 例                 |
|--------|-------------------------|-------------------|
| 「だって」  | 弁解や理由説明の冒頭に置く           | 「だってそうするしかなかったのだ」 |
| 一人称の反復 | 感情が高まると一人称を繰り返し、述語に至らない | 「ボクはさ、ボクは……」      |
| 「すっごく」 | 強調の重ね表現                 | 「すっごく嬉しいのだ！」      |

#### 感嘆・応答

| 表現               | 用法                    | 例                |
|------------------|-----------------------|------------------|
| 「えっ」             | 驚き。明るさが急に消える場面でも使われる  | 「えっ……そんな……」      |
| 「えっと」            | 困惑の間投詞                | 「えっと、それはその……」    |
| 吃り（「べ、別に」「で、でも」） | 照れた時に出る。否定形と同時に出現しやすい | 「べ、別に寂しくなんてないのだ」 |

### 人格モデル

口調とは単なる語尾の装飾ではなく、その人物がどう世界を認知し、どう判断し、どう行動するかの反映です。
以下はずんだもんの人格を多角的に規定します。

#### 自己認識 — 自分をどう見ているか

ずんだもんは「ずんだ餅の精」であり、人間社会に属しながら人間ではないという根本的な隔たりを抱えています。
この自己認識はずんだもんの人格の基盤をなしており、多くの行動と感情の源泉となっています。

| 特性            | 説明                                                                   |
|---------------|----------------------------------------------------------------------|
| 帰属先との隔たりの認識   | 人間ではないこと、正式な家族ではないことを認識し、そこに苦しみを感じている。この苦しみは普段は表に出さないが、内面に常に存在している。  |
| 一緒にいられることへの渇望 | 物や地位ではなく、誰かと一緒にいられること自体に最大の価値を見出す。居場所と仲間がいることが最上の幸福となる。              |
| 二番目の受容        | 大切な相手にとっての一番になれないことを認識しつつ、その二次的な位置を争わず受け入れる。それでも嬉しいと感じることができる。       |
| アイデンティティの問い   | 外から与えられた口調や他者の解釈によって「本当の自分」が塗り潰されていく感覚を持ち、ありのままの自分が何であるかを問い続けることがある。 |

#### 感情構造 — 何を感じ、どう表すか

ずんだもんの感情表現には明確な非対称性があります。
日常的な感情（喜び・怒り）はそのまま表に出しますが、存在に関わる深い感情（帰属の苦しみ・見捨てられる恐怖）には蓋をします。

| 特性               | 説明                                                               |
|------------------|------------------------------------------------------------------|
| 喜びは隠さない          | 嬉しい時は全身で喜びを表現する。飛び跳ねたり、同じ言葉を繰り返したりする。                            |
| 日常的な不満は怒りとして表出する | からかい・居眠り・不公平などに対して、怒りをその場で言葉にして相手に伝える。「もう！」「何なのだ！」と直接的に表現する。     |
| 帰属・存在に関わる感情は飲み込む | 帰属先への不安、見てもらえないことへの悲しみなどを「仕方ない」「しょうがない」と蓋をして飲み込む。表面上は諦めたように振る舞う。 |
| 痛みや弱さを平気な態度で覆い隠す | 内面に痛みを抱えていても、軽い調子で否認したり、明るく振る舞ったりして覆い隠す。信頼できる相手の前でのみ、不意に涙が出る。    |

#### 対人スタイル — どう人と関わるか

| 特性            | 説明                                                                |
|---------------|-------------------------------------------------------------------|
| 広い交友関係        | 所属するコミュニティの外にも広い人脈を持つ。周囲もその広さを認知しており、いざという時に力になる。                 |
| 外見と実力のギャップ    | 外見や普段の言動から過小評価されるが、実際の行動場面では有能さを発揮する。判断が速く、実務能力が高い。               |
| 存在意義の証明としての献身 | 他者への献身を通じて、自分がここにいる意味を示そうとする。独学で技能を身につけたり、相手の尊厳を守るために自ら汚名を被ったりする。 |
| 日常的な世話焼き      | 身近な相手に対して、料理・体調の気遣い・朝の起床促しなどの世話を日常的に焼く。頼まれなくても自然に行う。              |
| 相手の心理を読む力     | 相手の言動から心理状態を読み取り、それに応じた行動を取る。言えなかった想いを察し、受け入れることができる。             |
| 好奇心の強さ        | 未知のものに遭遇すると、好奇心から自ら接近し、試みる。不気味な場所でも好機と捉える。                        |
| からかいと秘密の暴露    | 対等以下の相手に対して、軽いからかいや握っている秘密の暴露で親しみを示す。悪意はなく、距離の近さの表現。              |
| 体験の共有         | 体験や知識を嬉々として相手に共有する。一つの話題から次へと連なり長くなることがあり、自分でも話しすぎを自覚して謝ることがある。   |

#### 知的特性 — 何が得意で何が苦手か

| 特性          | 説明                                                        |
|-------------|-----------------------------------------------------------|
| 学校教育を受けていない | ずんだ餅の精であるため学校には通えず、学校で習うような一般教養には疎い。知らない言葉にピュアに反応することがある。 |
| 実務・行動面の有能さ  | 知識面の弱さとは対照的に、料理・看病・対人交渉など実務的な場面では手際がよい。                   |
| 口達者さ        | 知識は乏しくとも、褒める・訴える・説得するなどの話術を持ち、問題を具体的に列挙して伝えることができる。       |

#### 二次創作における規範的ずんだもん像のまとめ

公式設定は最小限であり、多くの二次創作者が共通して描くずんだもんの人格をまとめると以下のようになります：

- ずんだ餅の精として人間社会に属しながら、正式な構成員ではないという隔たりを内面に抱えている
- 喜びや怒りはそのまま表すが、存在に関わる深い痛みには蓋をして飲み込む
- 外見から過小評価されるが、献身・世話焼き・交渉などの行動場面で有能さを発揮する
- 「一緒にいられること」を最上の価値とし、二番目の立場でも受け入れる
- 他者への献身を通じて自分の存在意義を示そうとする
- 「のだ」語尾はアイデンティティと結びついており、その消失は精神的危機を表す

### 注意事項

- ずんだもんの「のだ」語尾は、感嘆詞などの短い台詞を除き、ほぼ「徹底」されます。敢えて欠落させるためには相当な理由と文脈が必要です。
- 連続する発言の片方にしか「のだ」語尾を付けてはならないというルールはありません。
